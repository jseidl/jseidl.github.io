Multi-TOR

Multi-TOR is a shellscript to open multiple TOR instances each one with a different identity (IP address) and bound to a specific socket.

Port numbers will start at 9050 and grow up the number of instances required. From there you can bind the sockets to other sockets and perform requests thru the TOR network.

Download

Download the latest version of Multi-TOR directly from the github project page.
